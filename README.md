# SEF Form Generator

Privacy-first accommodation registration form generator for Portugal (Boletim de Alojamento).

## Features

- 🔒 **Privacy-first**: All data stays in your browser - nothing sent to servers
- 🌍 **Multilingual**: English, Portuguese, Russian, Ukrainian, Spanish, French, German
- 📄 **PDF generation**: Creates printable bilingual forms (Portuguese + selected language)
- 📎 **Document attachments**: Attach passport/ID photos directly in the PDF
- 📱 **Mobile-friendly**: Works on any device

## Development

```bash
npm install
npm run dev     # Start dev server at localhost:5173
npm run build   # Build for production (outputs to dist/)
```
